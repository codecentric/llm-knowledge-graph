# inputs/

Hier liegen die Quelldokumente, aus denen der Knowledge Graph extrahiert wird.

Formate: Markdown, CSV, TXT, JSON, DOCX (als Text)
Struktur: frei – Unterordner nach Thema, Datum oder Quelle

Dokumente bleiben nach der Extraktion erhalten, damit die Herkunft jedes
Graphknotens nachvollziehbar bleibt.
