Über dieses Projekt

Dieser Knowledge Graph modelliert die Fachdomäne eines fiktiven Online-Shops als RDF-Ontologie. Alle Daten liegen als Turtle-Dateien im Verzeichnis graph/ und werden direkt im Browser per SPARQL (Oxigraph WASM) abgefragt – ohne Server, ohne Backend.

Begleitcode zum Artikel „Knowledge Graphs als Langzeitgedächtnis für AI-Agenten"  ·  GitHub
